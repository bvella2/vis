Interactive Dashboard

Introduction

This is an Interactive Dashboard for Chicago Red Light Violations Dataset. The dataset is preprocessed and reduced to 100,000 rows as it is very large.
This dataset reflects the daily volume of violations created by the City of Chicago Red Light Program for each camera. The data reflects violations that occurred from July 1, 2014 until present, minus the most recent 14 days. There are multiple Aggregations and data manipulations in this project. Please go through them carefully.

1. Violation Map

This is a violation Map. Each dot here represents a Camera. We calcuated Yearly, Monthly and Weekly violations that each camera captured throughout the Years.
Click on the camera to get a line chart next to the map. Then ,Select an option for Time. You can see the changes in violations Yearly, Monthly and Weekly.

2. Aggregated Bar Graph

This is a Bar Chart That calculates Total Violations occured each Year, Month and Week.
From the previous visualization, if you change the options in dropdown, you can see that this bar chart and the above Line chart in the previous visualizationchange accordingly.

3. Brushable Scatter Plot

On the Scatterplot, when you draw, the total violations of selected intersections over time and the total violations made by rest of the intersections over time is plotted as a Donut Plot.
